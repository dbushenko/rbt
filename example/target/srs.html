Use cases

[UCS-01] User signs in

User: not authenticated user.

Preconditions:

  • user not authenticated;
  • Form UI-01: Sign in form is shown.

Postconditions:

  • User is authenticated;
  • Form UI-02: Home page is shown.

Main flow:

  1. The user specifies his credentials and signs into the system using UI-01/A01: Login.
  2. The system checks user credentials.
  3. The system authenticates the user.
  4. The system shows the form UI-02: Home page and shows the message Successfull authentication.

Alternative flows:

2a. If credentials are wrong then the system stops the use case and shows the message Wrong credentials.

References from use case

M-02, M-01, UI-02, UI-01.
